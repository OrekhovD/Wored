## 🏆 Рекомендации по использованию

### Free Only режим
**Цепочка:** `qwen-turbo → ai-minimax/minimax-m2.7 → qwen-plus`

- Использовать бесплатные модели
- Fallback на qwen-plus при необходимости
- Ограничение: нет glm-5

### Balanced режим
**Цепочка:** `qwen-turbo → qwen-plus → glm-5`

- Баланс между стоимостью и качеством
- glm-5 доступен при остатке квоты ≥10%
- Оптимально для большинства задач

### Premium режим
**Цепочка:** `glm-5 → qwen-plus → qwen-turbo`

- Лучшее качество в приоритете
- glm-5 всегда доступен
- Для критически важных запросов
