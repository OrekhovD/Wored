Важное: ключи не присылай сюда. Проверка должна выполняться локально Hermes/Qwen в WSL, с маскировкой.
Hermes поддерживает credential pools: несколько ключей одного провайдера можно ротировать, а при 429 он должен переключаться на следующий ключ; это лучше, чем вручную менять ключи. Также есть fallback providers: если основной провайдер падает, Hermes может переключиться на другую пару provider:model.
Для custom/OpenAI-compatible endpoint в Hermes используется универсальная схема provider, model, base_url, api_key; если задан base_url, Hermes обращается к этому endpoint напрямую.
MiniMax M2.7 через NVIDIA NIM доступен как minimaxai/minimax-m2.7, и NVIDIA описывает его как модель для complex software engineering, agentic tool use и office/document workflows.
